# ファビコン候補デザイン

このディレクトリには、勤怠管理キットアプリケーション用のファビコン候補が含まれています。
すべてSVG形式で作成されており、10種類の異なるデザインパターンから選択できます。

## デザインコンセプト

プラモデルのランナーフレーム（組み立て前のパーツ配置枠）をモチーフにしたデザインです。

### 構成要素

1. **外枠**: プラモデルのランナーフレームを模した四角い枠
2. **内枠**: 追加の内側フレーム
3. **バリ（ゲート）**: プラモデルのパーツをフレームから切り離す際の接続部分。上下左右に配置され、各デザインで位置が異なります
4. **中央の大きな時計**: 勤怠管理の時間要素を象徴
5. **中サイズのチェックマーク**: タスク完了や出退勤記録を表現
6. **小さな要素**:
   - 社員証（IDバッジ）
   - 太陽（日中勤務）
   - 月（夜勤）

## カラースキーム

- プライマリカラー: `#2563eb` (青) - 時計、社員証
- セカンダリカラー: `#64748b` (グレー) - フレーム
- 成功カラー: `#16a34a` (緑) - チェックマーク
- 太陽: `#fbbf24` (黄色)
- 月: `#94a3b8` (薄いグレー)
- ホワイト: `#ffffff` - 時計の文字盤

## プレビュー

すべてのファビコンを一度に確認するには、`preview.html`を開いてください。

## 使用方法

選択したファビコンを使用するには、`index.html`の`<link>`タグを更新してください：

```html
<link rel="icon" type="image/svg+xml" href="/favicons/favicon-XX-name.svg" />
```
