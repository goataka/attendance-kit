# ファビコンバリエーション

このディレクトリには、勤怠管理キットアプリケーション用のファビコン20パターンが含まれています。

## デザインコンセプト

各ファビコンには以下の要素が含まれています：

- **プラモデルのフレーム**: 外側と内側に配置（プロジェクトの構造を表現）
- **時計**: 大きく中央に配置（勤怠管理の核心）
- **チェックマーク**: 中程度のサイズ（承認・完了の意味）
- **社員証**: 小さな要素（社員識別）
- **太陽**: 小さな要素（日中の勤務）
- **月**: 小さな要素（夜間の勤務）

## カラーパレット

アプリケーションの基調カラーを使用：

- **Primary Blue**: `#2563eb`（主要色）
- **Secondary Gray**: `#64748b`（補助色）
- **Success Green**: `#16a34a`（成功・承認）
- **Accent Yellow**: `#fbbf24`（太陽）

## 使用方法

### HTMLでの参照

```html
<link rel="icon" type="image/svg+xml" href="/favicons/favicon-01.svg" />
```

### 推奨バリエーション

- **標準使用**: favicon-01.svg（バランスの取れたデザイン）
- **明るいテーマ**: favicon-06.svg（白背景）
- **グリーンアクセント**: favicon-02.svg（承認・成功を強調）
- **シンプル**: favicon-14.svg（ミニマルデザイン）

## カスタマイズ

各SVGファイルは編集可能です。色やサイズを変更する場合は、適切なSVGエディタ（例：Inkscape、Figma）を使用するか、テキストエディタで直接編集できます。
