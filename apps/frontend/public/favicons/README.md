# ファビコン候補デザイン

このディレクトリには、勤怠管理キットアプリケーション用のファビコン候補が含まれています。
すべてSVG形式で作成されており、20種類の異なるデザインパターンから選択できます。

## カラースキーム

- プライマリカラー: `#2563eb` (青)
- セカンダリカラー: `#64748b` (グレー)
- 成功カラー: `#16a34a` (緑)
- ダークプライマリ: `#1d4ed8` (濃い青)
- ホワイト: `#ffffff`

## デザインパターン

## プレビュー

すべてのファビコンを一度に確認するには、`preview.html`を開いてください。

## 使用方法

選択したファビコンを使用するには、`index.html`の`<link>`タグを更新してください：

```html
<link rel="icon" type="image/svg+xml" href="/favicons/favicon-XX-name.svg" />
```
