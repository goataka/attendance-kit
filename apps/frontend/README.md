# Frontend

フロントエンドアプリケーション

## 概要

勤怠管理システムのフロントエンドアプリケーションです。

## 開発

```bash
# 開発サーバーの起動
npm run dev -w @attendance-kit/frontend

# ビルド
npm run build -w @attendance-kit/frontend

# テスト
npm test -w @attendance-kit/frontend
```
