# 勤怠アプリ フロントエンド

勤怠管理システムのReactフロントエンドアプリケーション。

## 技術スタック

- React 18
- TypeScript
- Vite
- プライマリカラー: #007CC0

## 開発

```bash
# 開発サーバー起動
npm run dev:frontend

# ビルド
npm run build:frontend
```

## 機能

- 出勤打刻
- 退勤打刻
- 打刻履歴表示
- レスポンシブデザイン

## 環境変数

- `VITE_API_BASE_URL`: バックエンドAPIのベースURL（デフォルト: http://localhost:3000/api）
