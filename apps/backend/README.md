# Backend

バックエンドアプリケーション

## 概要

勤怠管理システムのバックエンドAPIです。

## 開発

```bash
# 開発サーバーの起動
npm run dev -w @attendance-kit/backend

# ビルド
npm run build -w @attendance-kit/backend

# テスト
npm test -w @attendance-kit/backend
```
