# Website

Webサイト

## 概要

勤怠管理システムの情報提供サイトです。

## 開発

```bash
# 開発サーバーの起動
npm run dev -w @attendance-kit/website

# ビルド
npm run build -w @attendance-kit/website

# テスト
npm test -w @attendance-kit/website
```
